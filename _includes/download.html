What is Lightbend Activator?

Lightbend Activator gets you started with the Lightbend Reactive Platform, including Akka, Play Framework and Scala. It is a hub for developers that want to build reactive applications, providing an in browser environment for creating new applications.

Getting started is a snap, please refer to the Getting Started page, including an introduction video.
